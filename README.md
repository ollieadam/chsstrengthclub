# CHS Strength Club Website

Static HTML site for Charleston Strength Club. Hosted on GitHub Pages.

**URL:** https://chsstrengthclub.com
**Git Remote:** `git@github.com:ollieadam/chsstrengthclub.git`

## Structure

| Path | Purpose |
|------|---------|
| `index.html` | Home page |
| `global.css` | Shared styles |
| `script.js` | Shared JS (nav, modals, forms) |
| `locations/` | Location pages |
| `blog/` | Blog posts |
| `docs/` | Build notes and architecture docs |

## Deploy

Push to `main` → auto-deploys via GitHub Pages.
